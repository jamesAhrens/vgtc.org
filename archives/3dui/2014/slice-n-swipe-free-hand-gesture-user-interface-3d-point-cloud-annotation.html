Authors: 
Felipe Bacim, Mahdi Nabiyouni, Doug A. Bowman
DOI Bookmark: 
http://dx.doi.org/10.1109/3DUI.2014.6798882
Keywords: 
Free-hand gesture, 3D interaction techniques, natural user interfaces, progressive refinement
Abstract: 
Inspired by the current generation of "natural user interface" technologies, we designed a set of tools for 3D point cloud annotation based on free-hand gesture input. Our techniques include a chef's knife metaphor to cut the point cloud and swipe away unwanted parts, a resizable volumetric cursor to specify complex 3D shapes, and a lasso tool to draw a selection outline. These tools are used interchangeably through progressive refinement to allow the user to specify points of interest and annotate. We will demonstrate a prototype of the system using a Leap Motion Controller and a 3D mouse.
